In order to see textures:
1. Downdload and install Python from www.python.org (make sure to add enviroment variables when installing)
2. Run this from the windows command line:
# in Python ver 2.x
python -m SimpleHTTPServer
# in Python ver Python 3.x
python -m http.server

This will serve files from the current directory at localhost under port 8000:

3. enter http://localhost:8000/
4. Copy the project folder to the current directory that is shown in http://localhost:8000/.
5. press on the project folder.
